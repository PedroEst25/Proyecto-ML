## Objetivo
El objetivo principal de este proyecto es diseñar, implementar y evaluar un pipeline completo de Machine Learning que abarque desde el preprocesamiento de datos y la clasificación supervisada, hasta el análisis de agrupamiento (clustering) no supervisado.

## Dataset Utilizado

**Online shoppers intention:** Es una coleccion de datos que ayuda a predecir si la sesión de navegación de un usuario en una página web de comercio electrónico terminará o no en una compra (transacción).

Enlace: https://www.kaggle.com/datasets/imakash3011/online-shoppers-purchasing-intention-dataset

## Modelos Utilizados

 1.**Decision Tree Classifier**
 
 2.**Maquina de soporte vectorial**
 
 3.**Random forest**
 
 4.**Reduccion de diemsionalidad**
